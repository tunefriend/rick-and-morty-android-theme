## Install (any Android)

1. Download the release zip (or clone this repo)
2. Unzip on the phone
3. **Photos / Gallery** → open a wallpaper → **Set as** Home / Lock / both
4. **Settings → Wallpaper & style** → pick a **green / teal** color from the wallpaper

### Icons

Stock launchers **do not** retheme every app icon. Use `icons/` for:

- Contact / profile photos  
- Custom shortcuts (Lawnchair, Nova, etc.)  

For full icon packs, use a third-party launcher + icon pack app.

## GrapheneOS

Same steps. Prefer **Photos** over **Files** if “Can’t open” appears.
